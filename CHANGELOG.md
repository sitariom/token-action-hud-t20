# Changelog

## [1.4.3] - 2026-03-16
### Adicionado
- Nova categoria **Condições** com suporte a ícones e status ativos.
- Categorização de **Magias** por Círculo.
- Categorização de **Poderes** por Subtipo (Classe, Geral, Origem, Tormenta, Destino).
- Nomes de Atributos e Perícias corrigidos com fallback manual e capitalização.
- IDs únicos para todas as ações (`attribute_`, `skill_`, `item_`, `condition_`) para compatibilidade total com HUD Core v1.4+.
- Propriedade `listName` adicionada para facilitar busca no editor do HUD.

## [1.3.0] - 2024-03-16
### Adicionado
- Suporte inicial para o sistema Tormenta20.
- Integração com Token Action HUD Core.
- Ações para Atributos, Perícias, Armas, Equipamentos, Consumíveis, Magias e Poderes.
