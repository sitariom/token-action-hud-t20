# Token Action HUD: Tormenta20

[Português](#português) | [English](#english)

## Português

Este módulo adiciona suporte para o sistema **Tormenta20** no módulo **Token Action HUD Core**.

### Funcionalidades
- Adiciona botões de ação rápidos para o token selecionado.
- Suporte para:
  - Atributos
  - Perícias
  - Armas
  - Equipamentos
  - Consumíveis
  - Magias
  - Poderes

### Instalação
1. Instale o módulo **Token Action HUD Core** no Foundry VTT.
2. Instale este módulo usando o link do manifesto:
   `https://github.com/sitariom/token-action-hud-t20/releases/latest/download/module.json`
3. Ative ambos os módulos no seu mundo.

### Requisitos
- Foundry VTT v11+
- Sistema Tormenta20
- Token Action HUD Core

## English

This module adds support for the **Tormenta20** system in the **Token Action HUD Core** module.

### Features
- Adds quick action buttons for the selected token.
- Support for:
  - Attributes
  - Skills
  - Weapons
  - Equipment
  - Consumables
  - Spells
  - Powers

### Installation
1. Install the **Token Action HUD Core** module in Foundry VTT.
2. Install this module using the manifest link:
   `https://github.com/sitariom/token-action-hud-t20/releases/latest/download/module.json`
3. Enable both modules in your world.

### Requirements
- Foundry VTT v11+
- Tormenta20 System
- Token Action HUD Core
